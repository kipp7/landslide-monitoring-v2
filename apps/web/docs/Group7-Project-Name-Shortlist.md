# 项目命名候选（第七组）

## 推荐名（首选）
- 山稳数境 | Mountain Insight
  - 含义：山体稳定 + 数据可视化意境；兼具技术与设计感
  - 传播：中英皆可，便于路演与提交材料标题

## 备选名
- 岩稳瞰 | RockGuard Visual
  - 强调“稳”和“瞰（全局洞察）”，安全与监测并重
- 坡安洞见 | Landslide Insight
  - 聚焦灾害“洞见”，偏分析与预警价值
- 陡安图谱 | Slope Atlas
  - 突出“图谱化呈现”，强调可视化方法
- 智感滑守 | SmartSlide Watch
  - “智感”体现智能认知，“滑守”表达守护之意
- 山稳可视 | MountainGuard UI
  - 清晰直白，易于评审快速理解

## 命名建议
- 简洁可记、两到四个字为佳；可附英文副标题
- 避免过于学术或过度工程化，突出设计与体验价值
- 标题用于：报名信息、作品题图、视频片头与PDF封面

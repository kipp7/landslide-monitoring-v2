# Task: hardware-productization

## Goal

把现有山体滑坡监测硬件从已联调的演示样机推进为可制造、可安装、可维护、可验证的现场设备，覆盖现场节点 PCB/外壳、RK3568 网关外壳、线束、电源和验收资料。

## Current State

- 现场节点 V1.2 已实机跑通双路 RS485：土壤传感器和倾角传感器。
- 当前实物使用 RK2206、DL-XLS1、CJMCU-752、两路隔离自动方向 RS485 和 3S 11.1 V 5000 mAh 电池。
- 已采购防水箱标称外形为 320 x 240 x 145 mm；卖家图实际约 317.9 x 238.9 x 144.6 mm，底壳深约 91.1 mm。箱内无高安装柱，仅有低位孔。
- 已采购 SWM-10W 太阳能板：290 x 240 x 17 mm、10 W、Vmp 18.68 V、Voc 22.30 V、0.67 kg。
- 当前 PCB Gerber 真值为 170 x 115 mm，不是布局文档中的 150 x 100 mm。
- 当前生产 Gerber/实物为模块版，而公开原理图仍是分立 SC16IS752/SP3485 版。
- 当前现场节点使用大型防水箱，但内部固定、线束、保险、充电、透气和维护设计未冻结。
- 已将产品化总纲、R1.3 规格、外壳、安装板、太阳能、采购和概念图整理到 `hardware/field-node/`。
- 已建立 `hardware/field-node/REVISION.md`，当前集成版本为 `FIELD-NODE-HW-EVT0.1`，尚未放行制造。
- 内部结构改为 `265 x 185 x 3 mm` FR4/G10 可拆安装板概念，孔位待实物转印/测量。
- 箱体低位孔当前口述约为 `Ø2 mm` 塑料底孔，商家说明配 `M3` 自攻螺丝；正式设计采用 FR4 `Ø3.5 mm` 通孔 + 限深自攻螺丝，孔深和是否贯通仍待实测。
- 当前载板已有负载电源开关，不重复增加箱内负载开关；该开关不代替太阳能充电管理，也不切断独立充电支路。
- 箱体内轮廓四侧有向内弧形槽/台阶，`265 x 185 mm` 只作为 FR4 最大外包络毛坯，正式板必须由实物模板生成避让缺口，不能直接加工完整矩形。
- 倾角变送器机械接口已由手册冻结：`90 x 58 x 36 mm`、4 x `Ø3.6 mm`、孔距 `78 x 39 mm`，底座孔中心为 `(6,9.5)/(84,9.5)/(6,48.5)/(84,48.5)` mm。比赛版传感器保留箱内，使用 `120 x 85 x 3 mm` 独立 304 小基准板；传感器 M3 螺纹孔中心为 `(21,23)/(99,23)/(21,62)/(99,62)` mm。FR4 下方四点支撑孔位仍需结合箱体实物确定。
- 当前太阳能充电采购改为完整 `CN3791` 3S 12.6 V 成品模块；`LT3652` 仅保留为后续自研充电 PCB 芯片路线，不采购裸芯片/贴片小板。

## Constraints

- 不能破坏已经验证的 RK2206 引脚和 A/B/C 固件接口。
- SC16IS752 当前模块地址为 0x4D、晶振为 1.8432 MHz。
- 当前模块化方案以手焊和快速维护为主，R1.3 不同时进行全分立化。
- 正式结构图必须基于实测尺寸和确定的箱体型号，照片估算不能作为开孔依据。
- 室外设备必须考虑进水、冷凝、日晒、浪涌、线缆拉力、振动和电池安全。

## Plan

- 收集箱体、电池、线缆、安装支架和所有模块的实测尺寸。
- 从 V1.2 LCEDA/实物重建模块版原理图和完整 BOM。
- 完成 R1.3 电源保护、传感器分路保护、真实电量采集和 DL-XLS1 硬复位设计。
- 完成现场节点背板布局、格兰头图、线束图和安装支架 CAD。
- 打样 R1.3，执行 EVT 电气/通信/恢复测试。
- 制作整机样机，执行 DVT 防水、温升、冷凝、振动、续航和维护测试。
- 单独完成 RK3568 + EC200A 网关箱的散热、天线和接口结构。

## Open Questions

- 防水箱低位孔的实测孔径、深度、贯通状态、配套 M3 自攻螺丝长度和 6 个可用孔中心坐标是什么？
- 电池 BMS、充电器、尺寸、重量和允许充放电温度是什么？
- 整机待机、采样、发射和最坏工况功耗分别是多少？当地最差月份的有效日照是多少？
- RS485 最大线长、走线方式和现场雷暴等级是什么？
- 比赛版倾角传感器保留箱内；现场长期版的最终测量基准和支架仍需冻结。
- 目标 IP 等级、环境温度、预算和计划产量是多少？
- RK3568、EC200A 与声光告警是否同箱？

## Done When

- PCB 原理图、PCB、Gerber、BOM、装配图和实物完全一致。
- 现场节点和 RK3568 各有可生产外壳/背板/开孔/线束图。
- R1.3 完成 EVT，整机完成 DVT，测试记录可追溯。
- 装配和维护不依赖未记录的手工经验。

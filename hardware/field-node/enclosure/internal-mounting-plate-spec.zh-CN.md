# 防水箱内安装板规格

状态：`DRAFT / 孔位实测前`

## 1. 设计目的

当前 IP65 防水箱内部没有高安装柱，只有若干低位孔。增加一块可拆安装板，把 PCB、电池托架、MPPT、保险和线槽统一固定到安装板，避免每个部件分别在箱底打孔。

## 2. 箱体已知尺寸

| 项目 | 尺寸 |
| --- | ---: |
| 标称外形 | 320 x 240 x 145 mm |
| 卖家图外形 | 317.9 x 238.9 x 144.6 mm |
| 底壳深度 | 约 91.1 mm |
| 盖体高度标注 | 约 53.5 mm |
| 内部多级轮廓 | 297.0109 x 218、276.0655 x 197、239.4 x 160 mm |
| 低位安装孔 | 约 `Ø2 mm` 塑料底孔，商家说明配 `M3` 自攻螺丝；孔深和是否贯通待实测 |

内部多级轮廓的具体含义需要实物确认，不能把其中任意一组直接当作可加工尺寸。

## 3. 安装板初始规格

- 初始外包络毛坯：`265 x 185 x 3 mm`，不是已经冻结的完整矩形外形。
- 首选材料：FR4/G10 阻燃环氧玻纤板，建议自然色或黑色，边缘倒钝。
- 箱体内边框四侧存在向内的弧形槽/台阶，正式安装板必须按实物加工对应的 U 形避让缺口；不得直接订购一块无避让的 `265 x 185 mm` 成品矩形板。
- 圆角和避让槽：按箱体圆角、侧壁台阶和螺钉操作空间实配，卖家截图不能提供完整半径和深度。
- 边缘间隙：相对 `276.0655 x 197 mm` 参考轮廓，理论每侧约 5.5/6 mm。
- 板面不得遮挡箱盖螺钉、密封槽、透气阀和底部格兰头操作区。

备选材料是 `2 mm 5052` 铝板。采用铝板时必须：

- 在 DL-XLS1/GPS 天线投影区开窗或把天线伸出金属板边缘。
- 对板边去毛刺并做绝缘隔离。
- 明确铝板与 PE/屏蔽地的连接，不能偶然接触逻辑 GND。

## 4. 与箱体连接

优先级如下：

1. 现有孔确认为 `M3` 自攻螺丝底孔且为盲孔：选 6 个分布均匀的孔；安装板加工 `Ø3.5 mm` 通孔，使用限深 `M3` 自攻螺丝、平垫和按高度实配的隔离套固定。自攻螺丝只咬合箱体塑料柱，不在 FR4 上攻螺纹。
2. 现有孔无螺纹但可装热熔/压入螺母：必须确认箱体材料和底厚，避免热损伤、开裂或穿底。
3. 现有孔为贯通孔：不直接使用普通螺钉；改用 EPDM 密封垫圈 + 密封螺钉 + 外部封帽，或使用箱内粘接式螺柱。
4. 现有孔不可用：使用与箱体材料兼容的粘接式 M4 底座螺柱，不在密封底面新增孔。

所有方案都必须经过拉力、冷热循环和喷水测试。普通双面胶、热熔胶和包装泡沫不作为永久结构固定。

箱体塑料自攻孔不作为日常维护螺纹。装配时控制扭矩且避免反复拆装；PCB、充电板和线槽均从安装板正面独立拆卸，正常维护不拆安装板。

## 5. 板上安装

- 现场节点 PCB：`170 x 115 mm`，4 x M3，孔中心距 PCB 边 3.5 mm。
- PCB 使用 `8-10 mm` M3 支柱，底面不得接触安装板。
- 电池使用独立阻燃托盘和两条绑带，托盘固定到安装板。
- 太阳能充电板、充电支路保护和可插拔电池连接器位于太阳能/电池线束入口附近，远离 GPS/DL-XLS1。当前载板已有负载开关，不重复安装外置负载开关。
- 倾角变送器只有在安装板和箱体共同构成可靠测量基准时才可固定在板上；否则单独固定到外部监测基准。
- 外部端子方向朝向箱体下侧格兰头，留出线弯曲和端子操作空间。

## 6. 孔位获取方法

正式 DXF 前先制作一块最大外包络 `265 x 185 mm` 的透明 PET 片或硬纸样板：

1. 样板放入箱内，沿四周弧形槽、圆角和侧壁台阶描线并逐步剪出避让缺口，确认能平放和取出。
2. 选择至少 6 个分布均匀的现有孔，转印孔中心；优先选择板内有完整承托面的孔柱。
3. 以安装板左下角为 `(0,0)`，测量每个孔的 X/Y 坐标，精度至少 0.5 mm。
4. 测量孔径、有效深度，并确认是盲孔还是贯通孔。
5. 测量底面到合盖后最低结构的净高。
6. 依据实测生成安装板 DXF、孔表和装配图。

## 7. 验收

- 安装板可在不拆密封圈的情况下装入和取出。
- 六个固定点全部锁紧后，板面无明显翘曲。
- 设备总成承受运输振动和任意方向 50 N 拉力后不位移。
- 螺钉未穿透未知薄壁，箱体无应力白化或裂纹。
- IP65 喷水测试后所有箱底固定点无渗水。
- PCB、端子、保险和电池均可独立维护，不需要拆除倾角基准件。

## 8. 倾角传感器刚性基准

当前倾角变送器机械接口已由手册冻结：外形 `90 x 58 x 36 mm`，4 x `Ø3.6 mm` 安装孔，孔距 `78 x 39 mm`。以传感器底座左下角为 `(0,0)`，四孔中心为 `(6,9.5)`、`(84,9.5)`、`(6,48.5)`、`(84,48.5)` mm。其读数代表传感器安装面的姿态，不应依赖塑料箱壁、FR4 安装板或可蠕变的胶垫作为测量基准。

现场长期版优先将倾角传感器移到监测桩/被测结构的独立金属基准件。为保持比赛样机外观和装配一致性，当前比赛版允许传感器留在防水箱内，但必须使用独立金属小基准板，不能直接安装到 FR4：

- 主安装板继续使用 3 mm FR4/G10，承载 PCB、电池、充电模块和线槽。
- 倾角传感器单独使用约 `120 x 85 x 3 mm` 的 304 不锈钢板；更轻的备选是 `120 x 85 x 4 mm` 5052 铝板。
- 金属小板加工 4 x M3 螺纹孔承接传感器。若 `120 x 85 mm` 小板与传感器同心，四个螺纹孔中心相对小板左下角为 `(21,23)`、`(99,23)`、`(21,62)`、`(99,62)` mm。小板自身另设 4 个安装孔固定到 FR4，禁止单边悬臂；这 4 个板间固定孔需要结合箱体支撑位置确定。
- FR4 在倾角小板下方增加 4 个刚性隔离柱/支撑点，并尽量落在箱体现有塑料孔柱附近，减少 FR4 局部弯曲。
- 箱体外部使用上下两组抱箍或四点背架固定；比赛搬运后重新校零并检查扭矩标记。
- 整块 `265 x 185 x 3 mm` 普通钢板约重 1.16 kg，不采用整块铁底板，以免塑料自攻柱长期受载，同时避免大面积金属影响 GPS/DL-XLS1 天线。

现场长期版结构仍为：

- 传感器直接用 4 x M3 螺钉固定到平整的 304 不锈钢基准板，建议板厚 3 mm；备选 4 mm 阳极氧化 5052 铝板。
- 基准板与监测桩使用刚性角码/槽钢连接；圆杆使用上下两组 U 型抱箍，轴向间距建议不小于 150 mm，并设置防转结构。
- 防水箱可以安装在同一外部背架上，但箱体塑料和 FR4 不进入倾角传力路径。
- 不在传感器与基准板之间垫普通橡胶、泡棉或厚胶层；安装面需要平整、贴合并在现场校零。
- 传感器为 IP65，外置时增加遮阳防雨罩，电缆出口朝下形成滴水弯。

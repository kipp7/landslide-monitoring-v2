# 倾角传感器机械接口规格

状态：`FROZEN / 来源为设备手册`

数据来源：`3458866111倾角变送器（485型）使用说明书.pdf` 第 5 页。

## 1. 传感器外形

| 项目 | 尺寸 |
| --- | ---: |
| 总长 | 90 mm |
| 总宽 | 58 mm |
| 总高 | 36 mm |
| 安装孔 | 4 x Ø3.6 mm |
| 横向孔距 | 78 mm |
| 纵向孔距 | 39 mm |

## 2. 传感器底座孔坐标

俯视安装底座，以底座左下角为 `(0,0)`，X 沿 90 mm 长边向右，Y 沿 58 mm 短边向上：

| 孔号 | X (mm) | Y (mm) |
| --- | ---: | ---: |
| H1 | 6 | 9.5 |
| H2 | 84 | 9.5 |
| H3 | 6 | 48.5 |
| H4 | 84 | 48.5 |

## 3. 比赛版金属小基准板

- 材料：3 mm 304 不锈钢；备选 4 mm 5052 铝板。
- 外形：`120 x 85 mm`，传感器在板上居中。
- 传感器固定：4 x M3 螺纹孔，中心坐标如下。

| 孔号 | X (mm) | Y (mm) |
| --- | ---: | ---: |
| T1 | 21 | 23 |
| T2 | 99 | 23 |
| T3 | 21 | 62 |
| T4 | 99 | 62 |

金属小板到 FR4 的四个独立固定孔不属于传感器手册尺寸，需结合箱体孔柱、FR4 异形轮廓和下方四点支撑位置后冻结。

## 4. 安装要求

- 传感器安装面与金属基准板紧密、平整、稳定。
- 传感器轴线与被测轴线平行，安装完成后现场校零。
- 不在传感器和基准板之间加入泡棉、厚橡胶或其他可蠕变软垫。
- 使用 M3 防松紧固件，装配后做扭矩标记。

# 现场节点外壳与整机结构规格

状态：`DRAFT / 实物尺寸待补齐`

## 1. 设计方向

现场节点继续采用成品工程塑料防水箱，不以亚克力夹板或 3D 打印外壳作为最终户外防护。结构设计对象是“整机箱”，包括电池、太阳能充电器、载板、倾角变送器、土壤变送器、线束、格兰头、透气阀和安装支架。

当前箱体标称外形已经冻结为 `320 x 240 x 145 mm`。卖家图给出的实际外形约为 `317.9 x 238.9 x 144.6 mm`，底壳深度约 `91.1 mm`，盖体高度标注约 `53.5 mm`。箱内没有高安装柱，只有一组低位孔。低位孔目前口述约为 `Ø2 mm` 塑料底孔，商家说明使用 `M3` 自攻螺丝；孔深、实际孔径和是否贯通仍需卡尺/深度尺确认。

卖家平面图还给出 `297.0109 x 218 mm`、`276.0655 x 197 mm`、`239.4 x 160 mm` 等多级轮廓尺寸。它们可能分别对应内壁、台阶和中央平面；图中四侧还有向内的弧形槽/凸台，所以正式底板不能按完整长方形加工，必须依据实物转印避让轮廓。

## 2. 防护与材料

- 目标防护等级：至少 IP65，长期室外建议 IP66/IP67。
- 箱体材料：抗 UV 的 PC/ABS 或玻纤增强聚酯；优先浅灰色。
- 外部增加遮阳/防雨帽，避免夏季日晒直接加热密闭箱体。
- 盖板采用连续密封圈和不脱落螺钉；现场维护后密封圈可检查、可更换。
- 使用 M12/M16 防水透气阀平衡压力并降低吸水呼吸效应；透气阀位于侧壁上部，不能正对雨水。
- 所有进线从箱体下侧进入，形成外部滴水弯。

## 3. 内部安装架构

使用可拆卸内安装板。初版建议以 `265 x 185 x 3 mm` 作为 FR4/G10 阻燃绝缘板的最大外包络，四边按箱体弧形槽和台阶加工避让缺口；该尺寸不是完整矩形下单尺寸。备选为 2 mm 5052 铝板，但金属板必须避开无线天线投影并可靠处理机壳地。PCB、电池和端子不得直接依赖箱体底部临时粘贴。

安装板优先利用现有低位盲孔固定，不新增箱底贯通孔。若确认是 `M3` 自攻螺丝底孔，FR4 加工 `Ø3.5 mm` 通孔并配平垫、限深自攻螺丝和隔离套；自攻螺丝只咬合箱体塑料柱。若现有孔是贯通孔，则必须使用 EPDM 密封垫圈、密封螺钉和外部封帽，或改用粘接式螺柱，完成 IP 测试后才能放行。

推荐分区：

```text
┌──────────── 320 mm ────────────┐
│ 电池区     MPPT/保护    倾角基准区 │
│ 充电保护     独立屏蔽区     刚性支架 │ 240 mm
│ 线槽/端子排                     │
│          现场节点载板            │
│ G1太阳能 G2土壤 G3倾角 G4维护/预留 │
└────────────────────────────────┘
                 箱体下侧
```

### 载板

- 载板实际 `170 x 115 mm`，使用 4 x M3 金属或阻燃尼龙支柱固定。
- 当前载板已有负载电源开关，不重复增加箱内负载开关；电池可插拔连接器兼作维修断开点。载板开关不切断太阳能充电支路。
- 支柱与背板采用防松螺母/螺纹锁固；不使用仅靠摩擦的黄铜长柱夹持整机。
- PCB 底面距背板建议 `8-12 mm`，最高器件到盖板净距至少 `15 mm`。
- 亚克力透明护板可保留为防触碰罩，但必须可单独拆卸，不能阻挡端子操作和天线净空。
- 载板四周至少留 20 mm 维修空间，端子插拔方向至少留 35 mm。

### 电池

- 电池使用阻燃托盘 + 两条机械绑带固定，托盘边缘限制四向移动。
- 电池与 PCB、尖锐螺钉和箱壁保持至少 10 mm 间隔；线束出口加软护套。
- 电池正极 100 mm 线长范围内设置保险；使用可触达、防反插、带锁的电池插头作为维修断开点。当前载板已有负载开关，不再重复增加外置负载开关。
- 电池下方不得使用普通包装泡沫作为永久固定；缓冲材料需阻燃、闭孔、耐温和耐老化。
- 电池标签和更换日期应在不开拆其他部件的情况下可见。

### 倾角变送器

- 倾角变送器必须固定到刚性基准板，安装面平面度和刚度优先于箱内美观。
- 比赛版为保持整机一致性，将倾角传感器保留在防水箱内：主板仍用 3 mm FR4，传感器单独固定到约 `120 x 85 x 3 mm` 的 304 不锈钢小基准板；小板四点固定，FR4 下方对应增加四点刚性支撑。
- 不采用整块钢质内安装板：`265 x 185 x 3 mm` 钢板约重 1.16 kg，会加重箱体塑料自攻柱负担，并可能影响 GPS/DL-XLS1 天线。
- 现场长期版仍推荐将倾角传感器外置到监测桩/被测结构的独立金属基准件，防水箱仅安装电子设备，不承担倾角测量基准。
- 传感器底座机械接口已由手册冻结：`90 x 58 x 36 mm`，4 x `Ø3.6 mm`，安装孔距 `78 x 39 mm`；底座左下角为原点时孔中心为 `(6,9.5)`、`(84,9.5)`、`(6,48.5)`、`(84,48.5)` mm。建议使用 3 mm 304 不锈钢基准板和 4 x M3 防松紧固件。
- 圆形监测桩采用上下两组 U 型抱箍，轴向间距建议不小于 150 mm，并增加防转结构；仅用一只抱箍不能作为倾角基准。
- 如果倾角用于测量坡体姿态，监测桩本身也必须刚性锚固到被测体；会晃动的普通设备立杆不能代替监测基准。
- 安装方向、X/Y/Z 轴、零点方向和紧固扭矩在支架和 SOP 中标注。

## 4. 线束与接口

- 每条外部线缆使用与外径匹配的 IP68 格兰头；禁止一个格兰头穿多根尺寸不匹配的圆线。
- 线缆进入端子前先固定到线槽/扎线座，端子不承受拉力。
- 多股线使用冷压管形端头，所有线束两端使用耐候标签。
- 电源线和 RS485 线分槽；开关电源输入与 GPS/DL-XLS1 天线区保持距离。
- 屏蔽线在箱体入口处就近处理，避免长屏蔽尾线穿过数字区。
- 保留 20%-30% 线槽和格兰头余量，不保留无固定的大圈盘线。

推荐格兰头定义：

| 编号 | 用途 | 备注 |
| --- | --- | --- |
| G1 | 充电/外部供电 | 与电池充电方案联锁、防误插 |
| G2 | 土壤传感器 | 4 芯电源 + RS485 |
| G3 | 倾角传感器 | 4 芯电源 + RS485 |
| G4 | GPS/天线或备用 | 不用时安装盲塞 |
| G5 | 维护接口（可选） | 默认封闭，不暴露裸 TTL |

## 5. RF、热与冷凝

- DL-XLS1 天线前方不放电池、倾角金属外壳、铝背板和金属支柱。
- 若使用金属背板，在天线投影区开窗或把天线端伸出背板边缘。
- GPS 天线尽量靠箱体上部，保持可见天空方向；不能贴在电池或金属传感器下方。
- MPPT/降压充电器属于开关电源噪声源，应远离 GPS、DL-XLS1 和 SC16IS752，并缩短太阳能输入与电池充电回路。
- 密闭箱内不采用普通开放风扇。先通过低功耗、器件间距、浅色箱体、遮阳罩和导热路径控制温升。
- 箱体最低点不布置 PCB 或裸端子；增加冷凝滴落隔离和可更换干燥剂位置，但干燥剂不能替代透气和密封设计。

## 6. 外部安装

- 箱体背面使用不锈钢立杆抱箍或墙装支架，不直接用箱体塑料底板承担倾角基准载荷。
- 太阳能板 `290 x 240 x 17 mm / 0.67 kg` 使用独立金属支架固定到立杆或共同背架，不直接把风载和重量传给箱盖。
- 若太阳能板位于箱体上方作为遮阳板，板与箱盖保持至少 `25-40 mm` 净距，并保证箱盖螺钉可拆、线缆可形成滴水弯。
- 太阳能板长边相对 320 mm 箱体只剩两侧各约 15 mm，短边与 240 mm 箱体相同；不能依赖箱盖边缘余量设置普通穿透螺栓。
- 所有外部紧固件采用 304/316 不锈钢并防松；异种金属接触考虑电化学腐蚀。
- 箱体离地、避开积水和落石路径；线缆先向下形成滴水弯再进入格兰头。
- 需要防雷的场景设置独立接地/浪涌保护边界，不能把雷电流引入 PCB 逻辑 GND。

## 7. 正式 CAD 所需尺寸

1. 现有低位孔实测直径（当前口述约 `2 mm`）、深度、是否贯通，以及选定 6 个固定孔相对箱内左下基准的中心坐标；同时确认商家配套 `M3` 自攻螺丝的长度。
2. `265 x 185 mm` 样板能否在不挤压内壁、圆角和密封结构的情况下平放和取出。
3. 电池长宽高、线束出口方向、重量和最小弯曲半径。
4. 载板装配后最大高度，包括 RK2206、SC16IS752、RS485 模块和透明护板。
5. 倾角变送器孔距、底座厚度、出线方向；现有手册外形约 90 x 58 x 36 mm。
6. 土壤变送器盒体和电缆接头尺寸。
7. 所有外部电缆直径、数量和最小弯曲半径。
8. 箱体在现场的立杆直径/墙面孔距和安装方向。
9. 太阳能板边框安装孔、背面接线盒外形/高度、出线位置和电缆外径。

## 8. DVT 验收

- 箱体关盖后无器件、线束或电池挤压密封圈。
- 线缆拉力 50 N（或按选定格兰头规格）后端子无受力、密封无位移。
- IPX5/IPX6 喷水后箱内无可见进水，绝缘和功能正常。
- 高低温循环后无冷凝滴落到 PCB；箱内最高温度不超过最低额定器件限制。
- 运输振动/跌落后电池、PCB、传感器和端子不松动。
- 倾角零点在安装、关盖、振动测试前后漂移满足项目阈值。
- 维护人员可在不拆电池和倾角基准件的情况下更换载板、保险和外部线缆。

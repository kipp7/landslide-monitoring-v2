# 现场采集节点 PCB R1.3 改版规格

状态：`DRAFT / 原理图重建输入`

## 1. 改版目标

R1.3 以已经跑通的 V1.2 模块拓扑为基础，解决资料不一致、电源保护不足、无线模块不可硬复位、真实电量缺失和现场接线不可维护的问题。R1.3 不以缩小尺寸为主要目标。

## 2. 必须保持的兼容性

- RK2206 插接板及现有 A/B/C 固件接口不变。
- `PB2/PB3` 保持 DL-XLS1 用户串口。
- `PB4/PB5` 保持 SC16IS752 I2C。
- `PB6/PB7` 保持 GPS UART。
- SC16IS752 模块当前真值：地址 `0x4D`、晶振 `1.8432 MHz`。
- 两路隔离自动方向 RS485 保持独立；CH1 土壤、CH2 倾角。
- 现场传感器端子继续提供 `VBAT_SW / GND / A / B`。

## 3. 原理图重建

必须从 V1.2 实物和 LCEDA 工程重建可读原理图，不再沿用分立 SP3485 的旧 PDF。原理图至少分为以下页：

1. 电池输入、保护、开关、充电/维修接口和电源树。
2. RK2206 插接接口、调试口和测试点。
3. DL-XLS1、GPS、可控复位/电源。
4. SC16IS752 模块和双路隔离 RS485。
5. 外部端子、屏蔽/机壳地、ESD/浪涌保护。

每个模块必须记录：制造商、完整型号、模块版本、供电范围、逻辑电平、引脚 1、实测尺寸和替代料规则。

## 4. 电源与电池

### 必做

- 电池输入采用防反插、带锁或防误插连接器，额定电流不低于 3 A。
- 在电池连接器之后、其他负载之前设置可更换保险丝或电子保险；额定值在实测峰值电流后冻结。
- 增加反接保护 MOSFET、输入 TVS、输入电解/陶瓷去耦。即使成品电源模块声称带保护，也不能在无规格书的情况下删除板级保护。
- CH1、CH2 的 `VBAT_SW` 分别增加可恢复保险或限流开关，单个传感器短路不得拖垮整机。
- 真实采集电池电压。优先使用 RK2206 可用 ADC + 高阻分压 + RC + 钳位；若无可靠 ADC 引脚，增加独立 I2C 电量计/ADC。
- 预留电池温度输入或数字温度传感器接口，低温充电和高温放电策略由系统定义。
- 明确充电口与系统供电口是否共口；禁止把普通 DC 圆孔直接作为无防护的电池充电接口。
- 太阳能输入先进入独立 3S 锂电 MPPT 充电器，再进入带 BMS 的电池；载板不得直接承受太阳能板 `22.30 V` 开路电压。
- 预留充电状态、太阳能输入电压/电流或至少充电存在信号，便于平台判断“低电量”与“无日照/充电故障”。

### 设计值待实测冻结

- 输入范围：建议按 `8.4-13.0 V` 设计，当前使用范围约 `9.0-12.6 V`。
- 保险丝：初始评估 `2 A`，以启动峰值、传感器短路和线束温升测试为准。
- 电池分压：满量程需覆盖至少 13.0 V，并满足 RK2206 ADC 绝对最大值和采样阻抗要求。

## 5. 无线与恢复

- DL-XLS1 增加独立负载开关或高边 MOSFET，由 RK2206 空闲 GPIO 控制。
- 增加硬件默认上电、GPIO 悬空安全态和手动测试点；软件异常不得永久关闭无线电源。
- 若 DL-XLS1 有可靠 RESET/KEY 复位时序，预留复位控制；否则用电源周期实现强恢复。
- DL-XLS1 天线正下方及前方保持无铜、无走线、无螺柱、无电池和无大面积金属。
- GPS 天线/模块远离开关电源、电池大电流线和 DL-XLS1 天线。

## 6. RS485 与现场保护

- 保持两路电气隔离；确认隔离模块在 `3.3 V` 供电下的 TX/RX 高低电平和实际隔离耐压。
- 每路端子附近增加现场侧 TVS。浪涌等级不足时，外置一级防雷器，PCB 只承担二级保护。
- 每路预留可配置 120 ohm 终端；默认装配状态由现场总线拓扑决定，不允许所有节点默认同时终端。
- 端子增加独立 SHIELD/PE，逻辑 GND 与屏蔽地不得无条件直连；预留 RC/电容/放电器件网络。
- 端子丝印同时标出 `V+ / GND / A / B / CH1/CH2`，不得只依赖电线颜色。
- 采用插拔式端子或保留当前弹簧端子，但必须满足线径、拉脱力和维护空间。

## 7. PCB 机械与布局

- R1.3 初版保持 `170 x 115 mm`，避免在电气风险未收敛时同时改机械包络。
- 四角安装孔以 Gerber 真值为基准：`3.2 mm` NPTH，中心距边 `3.5 mm`。下一版建议将铜禁布扩大到孔边至少 1.5 mm，并确认垫圈空间。
- 所有现场端子尽量集中在同一板边，面向箱体下方格兰头区。
- 电源区、数字区、RF 区和现场浪涌区明确分区；RS485 现场侧与逻辑侧保持模块规定爬电/电气间隙。
- 开关电源回路最小化；不允许大电流线穿过 GPS/DL-XLS1 天线区。
- 板上增加版本号、输入电压、端子定义、保险规格、二维码/序列号和危险极性标识。
- 测试点至少包含：电池输入、保险后、VBAT_SW、5V、3V3、GND、I2C、DL-XLS1 UART、GPS UART、两路 RS485 模块 UART。

## 8. BOM 与生产资料

R1.3 放行包必须同时包含：

- 可编辑原理图与 PCB 源文件。
- PDF 原理图、Gerber、钻孔、IPC-356/网表（工具支持时）。
- 完整 BOM：制造商料号、供应商料号、封装、数量、DNP、替代料、手焊/贴装属性。
- 坐标文件、装配图、模块安装方向图和首件检验表。
- 每个成品模块的采购链接、规格书快照、实物尺寸和版本照片。
- 固件硬件配置矩阵，防止 `0x4D/1.8432 MHz` 被其他模块版本覆盖。

## 9. EVT 验收

- 反接不损坏；恢复正确极性后可正常启动。
- `9.0 V`、`11.1 V`、`12.6 V` 输入均能稳定运行。
- CH1/CH2 分别短路时整机不复位，故障解除后通道恢复。
- DL-XLS1 可由 MCU 执行一次完整断电重启并恢复通信。
- 真实电池电压误差不大于 `±2%` 或 `±100 mV`（取更严者）。
- 两路 RS485 同时运行 72 小时无丢通道、死锁或异常复位。
- 端子、模块、板面最高温升和待机/工作电流有记录。
- 原理图、BOM、PCB 丝印与实物逐项一致。

# Field Node Power

[`solar-power-system-spec.zh-CN.md`](solar-power-system-spec.zh-CN.md) records the SWM-10W panel, 3S battery, charging architecture, energy budget, interfaces, and qualification tests.

No charging module is approved for production. A candidate must prove 3S Li-ion `12.6 V` CC/CV behavior, reverse-current blocking, acceptable quiescent current, thermal performance, and seven-day outdoor energy balance.

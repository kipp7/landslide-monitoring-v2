# Competition Docs Hub

本目录用于沉淀与比赛申报、作品定位、材料归类、数据资产表达、答辩口径有关的可复用文档。

当前入口：

- `2026-softcom-application-master-draft.md`
  - 可直接拆分为申报书、答辩稿、创业计划书的总稿母版
- `2026-softcom-work-positioning-draft.md`
  - 2026 软通赛题作品名称、简介、年度升级表达
- `2026-softcom-technical-route-reverse-evaluation.md`
  - 从评委、工程和创业视角反向评估技术路线该保留、该增加、该克制和该砍掉的内容
- `2026-softcom-material-index.md`
  - 软通赛题当前建议使用的源材料总索引与分类清单
- `2026-softcom-data-asset-strategy.md`
  - 滑坡数据库源、数据治理、模型数据资产、创业化方向说明

使用方式：

- 写申报书时先看：
  - `2026-softcom-application-master-draft.md`
  - `2026-softcom-work-positioning-draft.md`
  - `2026-softcom-data-asset-strategy.md`
- 找历史可复用材料时先看：
  - `2026-softcom-material-index.md`
- 后续若补：
  - 摘要版
  - PPT 首页版
  - 路演讲稿版
  - 创业计划书版
  应继续落到本目录，而不是散回聊天记录。

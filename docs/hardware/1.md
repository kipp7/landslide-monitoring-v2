  V1 先不要画整圈 38Pin。
  也不要走 E53。
  先直接用小凌派底边那排已经够用的 10Pin 外扩口来做。

  这排丝印我已经从板卡图和原理图里对上了，顺序就是：

  5V / GND / 3V3 / GND / SPI_MISO / SPI_MOSI / GPIO5 / UART_TX / UART_RX / GPIO4

  它刚好覆盖你现在项目要用的全部信号：

  - PB3 = SPI_MISO
  - PB2 = SPI_MOSI
  - PB5 = GPIO5
  - PB7 = UART_TX
  - PB6 = UART_RX
  - PB4 = GPIO4

  也就是：

  - PB2/PB3 给 XL01
  - PB4/PB5 给 SHT30 + MPU6050
  - PB6/PB7 给 GPS

  所以你现在原理图最稳的画法，是把这块板画成：

  “插到底边 10Pin 外扩口上的专用扩展板”

  这样比一上来画整圈 38Pin 简单很多，而且不改代码。

  三、P_RK2206_EXT10 这个连接器怎么画

  你在立创EDA标准版里，先放一个 1x10 2.54mm 连接器符号。

  名字改成：

  - P_RK2206_EXT10

  然后把这 10 个脚定义成下面这样：

| Pin号 | 小凌派丝印 | 实际GPIO/功能 | 你项目里的用途 |
| ----- | ---------- | ------------- | -------------- |
| 1     | 5V         | 板上5V        | 先不接，保留   |
| 2     | GND        | 地            | 接系统地       |
| 3     | 3V3        | 板上3.3V      | 先不接，保留   |
| 4     | GND        | 地            | 接系统地       |
| 5     | SPI_MISO   | PB3           | 接 XL01_RX     |
| 6     | SPI_MOSI   | PB2           | 接 XL01_TX     |
| 7     | GPIO5      | PB5           | 接 I2C_SCL     |
| 8     | UART_TX    | PB7           | 接 GPS_RX      |
| 9     | UART_RX    | PB6           | 接 GPS_TX      |
| 10    | GPIO4      | PB4           | 接 I2C_SDA     |

  这里最重要的点：

  - Pin1 和 Pin3 你现在不要接到你自己的 5V_SYS/3V3_SYS
  - 因为你前面已经定了：
      - RK2206 自己 Type-C 供电
      - 扩展板自己 J_PWR 供电
      - 两边只共地
  - 所以：
      - Pin2 和 Pin4 接 GND
      - Pin1 和 Pin3 先留空，或者加 NC 标记

  这样就不会反向给开发板送电。

  七、原理图怎么连，直接照着连

  这是你最关键的一步，我给你写死。

  1. 小凌派对 XL01

  - J_XL01.Pin3 (XL01_TX) -> P_RK2206_EXT10.Pin6 (PB2 / UART2_RX_M1)
  - P_RK2206_EXT10.Pin5 (PB3 / UART2_TX_M1) -> J_XL01.Pin4 (XL01_RX)

  也就是：

  - Pin6 -> J_XL01.Pin3
  - Pin5 -> J_XL01.Pin4

  2. 小凌派对 GPS

  - P_RK2206_EXT10.Pin9 (PB6 / UART_RX) <- GPS_TX
  - P_RK2206_EXT10.Pin8 (PB7 / UART_TX) -> GPS_RX

  也就是：

  - Pin9 -> J_GPS.Pin3
  - Pin8 -> J_GPS.Pin4

  3. 小凌派对 I2C

  - P_RK2206_EXT10.Pin10 (PB4 / GPIO4) -> I2C_SDA
  - P_RK2206_EXT10.Pin7 (PB5 / GPIO5) -> I2C_SCL

  再分给两个模块：

  - I2C_SDA -> J_SHT30.Pin3
  - I2C_SDA -> J_MPU6050.Pin3
  - I2C_SCL -> J_SHT30.Pin4
  - I2C_SCL -> J_MPU6050.Pin4

  4. 地

  - P_RK2206_EXT10.Pin2 -> GND
  - P_RK2206_EXT10.Pin4 -> GND
  - 所有模块 GND 都接 GND

  5. 模块供电

  - J_XL01.Pin1 -> 3V3_SYS
  - J_GPS.Pin1 -> 5V_SYS
  - J_SHT30.Pin1 -> 3V3_SYS
  - J_MPU6050.Pin1 -> 3V3_SYS

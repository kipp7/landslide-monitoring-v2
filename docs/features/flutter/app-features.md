# App 功能模块

## 功能列表

### 1. 认证模块 (auth)
- 登录页面
- 自动登录（Token 缓存）
- 退出登录
- 修改密码

### 2. 首页 (home)
- 设备概览（在线/离线数量）
- 告警概览（待处理告警数）
- 风险分布图
- 最近告警列表
- 快捷入口

### 3. 设备管理 (device)
- 设备列表（支持搜索、筛选）
- 设备详情
- 设备实时状态
- 设备位置（地图）
- 设备配置查看

### 4. 实时监测 (monitoring)
- 实时数据展示
- 数据趋势图表
- GPS 位移分析
- 历史数据查询
- 数据导出（生成报告）

### 5. 告警管理 (alert)
- 告警列表
- 告警详情
- 告警处理（确认/解决/忽略）
- 告警统计
- 推送通知接收

### 6. 地图展示 (map)
- 设备分布地图
- 设备状态标记
- 点击查看设备信息
- 风险区域标注

### 7. 设置 (settings)
- 个人信息
- 通知设置
- 告警订阅
- 主题切换
- 关于

## 页面流程

```
启动页
  │
  ├─ 未登录 ──▶ 登录页 ──▶ 首页
  │
  └─ 已登录 ──▶ 首页
                  │
    ┌─────────────┼─────────────┐
    │             │             │
    ▼             ▼             ▼
  设备管理      实时监测       告警管理
    │             │             │
    ▼             ▼             ▼
  设备详情      数据图表      告警详情
```

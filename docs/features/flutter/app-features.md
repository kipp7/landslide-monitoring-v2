# App 功能模块

## MVP 范围（Android v1）

- 登录/权限
- 站点（列表 + 地图）
- 设备（列表/详情/状态/趋势）
- 告警（列表/详情/ACK/RESOLVE）
- 巡查上报 / SOS
- 传感器字典驱动显示（`/sensors`）

## 功能列表

### 1. 认证模块 (auth)
- 登录页面
- 自动登录（Token 缓存）
- 退出登录
- 修改密码

### 2. 首页 (home)
- 设备概览（在线/离线数量，MVP）
- 告警概览（待处理告警数，MVP）
- 最近告警列表（MVP）
- 快捷入口（MVP）
- 风险分布图（可选，后续）

### 3. 站点 (stations)
- 站点列表（支持搜索、筛选，MVP）
- 站点地图（站点坐标标注，MVP）
- 站点详情（MVP）
- 站点关联设备列表（MVP）
- 风险区域标注（可选，后续）

### 4. 设备管理 (devices)
- 设备列表（支持搜索、筛选，MVP）
- 设备详情（MVP）
- 设备实时状态（MVP）
- 设备趋势图表（短时间窗口，MVP）
- 历史数据查询（可选，后续）
- 设备配置查看（可选，后续）

### 5. 告警管理 (alert)
- 告警列表（MVP）
- 告警详情（MVP）
- 告警处理（ACK/RESOLVE，MVP）
- 告警统计（可选，后续）
- 推送通知接收（可选，后续）

### 6. 设置 (settings)
- 个人信息（MVP）
- 退出登录（MVP）
- 通知设置（可选，后续）
- 告警订阅（可选，后续）
- 主题切换（可选，后续）
- 关于（可选，后续）

## 页面流程

```
启动页
  │
  ├─ 未登录 ──▶ 登录页 ──▶ 首页
  │
  └─ 已登录 ──▶ 首页
                  │
    ┌─────────────┼─────────────┐
    │             │             │
    ▼             ▼             ▼
  站点/地图     设备管理      告警管理
    │             │             │
    ▼             ▼             ▼
  站点详情      设备详情      告警详情
```

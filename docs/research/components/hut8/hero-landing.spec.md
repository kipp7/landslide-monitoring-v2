# Hut8 Hero Landing Specification

## Overview

- 参考位置：
  - Hut 8 landing manifesto
- 当前映射：
  - `apps/promo-demo` 首页第一屏
- 交互模型：
  - sticky hero
  - 内容随滚动停留较长时间

## 必备元素

- 左上角小字号品牌标记
- 一句更短的英文宣言
- 一个巨大主标题
- 一段较短 lead
- 右侧 summary / metrics card
- 底部横向关键词带

## 视觉重点

- 标题必须压住画面
- 右侧 summary card 负责给出第一层可读信息
- 不能像普通 SaaS hero 一样变成居中卡片页

## 当前产品翻译

- 英文宣言：
  - `Where Signal Unlocks Response`
- 中文主标题：
  - 数字山体空间宣言
- 右侧 summary：
  - 当前聚焦章节
  - 两个 CTA
  - 三个 metrics

## 不要做成

- 居中 banner
- 单纯左右两栏 marketing hero
- 没有固定背景舞台的纯文案首屏

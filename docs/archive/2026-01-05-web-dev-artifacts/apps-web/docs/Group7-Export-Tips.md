# Group 7 导出与打包提示

## 截图
- 浏览器 100% 缩放，关闭书签栏/控制台
- 统一浅/深色主题与背景；启用同一字体渲染
- 固定窗口宽度（≥1920px），必要时使用“响应式设计模式”
- 数据保持稳定（避免实时刷新导致曲线跳动）。必要时录制前刷新并等待 3–5 秒

## 视频
- 建议：OBS 1080p, 30fps, H.264，码率 6–10 Mbps；音频 AAC 128 kbps
- 鼠标移动缓慢，点击前停顿 300–500ms；使用系统光标捕捉
- 旁白脚本见《Group7-Video-Script.md》，字幕简洁居下

## PDF（交互流程）
- 打印本仓库《Group7-Interaction-Flow.md》为 PDF
- 页边距“窄”，纸张 A4；导出后检查中文字体嵌入

## 命名与结构
- 图片命名：`g7_[route]_[state]_[index].png`
- 打包结构参考《Group7-Submission-Checklist.md》

## 合规自检
- 不含第三方受限素材；或已取得授权并标注来源
- 隐私数据脱敏（设备 ID、地理坐标如需隐藏则马赛克）
- 与报名表、授权协议信息一致（人名、团队、赛区）

# 第七组（数字媒体设计高校学生组）提交清单

- 截止时间：2025-09-30 17:00（提交后不可修改）
- 赛道/组别：创意赛道 · 第7组 数字媒体设计高校学生组
- 项目：滑坡监测可视化与交互设计（基于本仓库前端）

## 1) 必交材料
- 高保真界面图 PNG（关键页面与核心交互状态）
  - 参考《Group7-Screenshot-Plan.md》的命名与必拍清单
  - 分辨率建议：1920×1080 或更高，统一背景与主题
- 交互流程 PDF（信息架构 + 关键用户流程）
  - 可由《Group7-Interaction-Flow.md》导出为 PDF（打印为 PDF）
- 演示视频 MP4（≤2 分钟，含操作说明与关键交互）
  - 参考《Group7-Video-Script.md》脚本与分镜
  - 建议：1080p，H.264，码率 6–10 Mbps，AAC 128 kbps
- 适配设备标注
  - 在 PDF 或图片标题中注明：PC / 移动端 / 大屏（如有）
- 报名表与作品授权使用协议（全体成员手写签名后上传）

## 2) 可选加分（建议）
- 设计源文件（Figma/Sketch）或设计规范 PDF（色板、字号、网格）
- 数据可视化与算法亮点 1 页附录（图表示例 + 关键指标）
- 无障碍与弹性布局说明（暗色/高对比度、键盘可达性）

## 3) 推荐目录结构（打包）
```
submit-g7/
  01_UI/                         # 界面 PNG（按命名规范）
  02_Flow_PDF/                   # 交互流程 PDF（由 .md 导出）
  03_Video/                      # 演示视频 MP4（≤2 分钟）
  04_Forms/                      # 报名表与授权协议（手写签名后扫描）
  05_Extras/                     # 源文件/设计规范/附录（可选）
  README.txt                     # 简要指引（可选）
```

## 4) 提交前自检清单
- [ ] PNG 命名、分辨率、主题风格一致
- [ ] PDF 含站点地图与主要任务流程（含适配设备标注）
- [ ] 视频 ≤120 秒，画面清晰、字幕简洁，重点流程完整
- [ ] 个人/团队信息与赛区正确，报名表与协议均已签字
- [ ] 不含受限信息或第三方版权素材（或已注明授权）

- /login 登录

- /analysis 分析页（异常类型/风险指标）
- /system-monitor 系统监控（设备健康/实时状态/告警）
- 可选补充：/optimized-demo（若展示关键优化点）

# 任务简报区

## 目标

本目录用于让总协调器给每个任务持续下达“当前任务简报”。

每个窗口以后不必反复从聊天里找上下文，只需要：

1. 打开对应的任务简报
2. 查看当前目标、约束、依赖和下一步
3. 再打开对应的任务报告继续工作

## 当前任务简报

- `docs/unified/agent-briefs/desk-api-align.md`
- `docs/unified/agent-briefs/gnss-protocol.md`
- `docs/unified/agent-briefs/algo-inventory.md`

## 协作方式

- 总协调器负责更新本目录下的任务简报
- 执行窗口负责更新 `docs/unified/reports/*.md`
- 如果任务发生切换、升级或阻塞，优先更新任务简报

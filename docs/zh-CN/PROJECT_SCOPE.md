# 项目范围

这个仓库保留原始 `landslide-monitoring-v2` 的 Git 历史，但当前公开目录只聚焦仍在维护的 Windows 桌面客户端。

## 当前公开支持范围

- `apps/desktop-ui/` - React + Vite 桌面监测界面。
- `apps/windows-shell/` - WPF + WebView2 Windows 宿主和安装器资源。
- `scripts/desktop/` - 桌面端开发、打包和验证脚本。
- `docs/` - 公开架构、发布、维护和中英文文档。
- `.github/` - CI、Issue 模板、Pull Request 模板和依赖自动化配置。

## 仅保留在历史中

以下内容不属于当前公开支持范围，已从当前目录树移除：

- Web 管理端。
- 移动端客户端。
- 后端服务。
- 部署基础设施。
- 硬件实验和板端调试记录。
- 内部日志、memory 文件、本地证据包和私有现场配置。

这些内容可能仍存在于旧提交中，用于追溯项目演进；不应视为仍在维护、可支持或可直接公开复用的模块。

## 维护规则

新工作默认保持桌面端聚焦。除非维护者明确决定恢复其他产品面，否则不要重新引入 Web、移动端、后端或基础设施目录。如果未来恢复某个模块，应作为有独立 CI、文档和安全审查的明确模块重新加入。

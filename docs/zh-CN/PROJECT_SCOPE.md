# 项目范围

这个仓库作为 Landslide Monitoring V2 的公开 Windows 桌面客户端维护。当前 `main` 分支聚焦桌面 UI、原生 Windows 宿主、打包脚本和公开文档。

## 当前维护范围

- `apps/desktop-ui/` - React + Vite 桌面监测界面。
- `apps/windows-shell/` - WPF + WebView2 Windows 宿主和安装器资源。
- `scripts/desktop/` - 桌面端开发、打包和验证脚本。
- `docs/` - 公开架构、发布、维护和中英文文档。
- `.github/` - CI、Issue 模板、Pull Request 模板和依赖自动化配置。

## 当前非目标

以下内容不属于当前维护范围：

- Web 管理端。
- 移动端客户端。
- 后端服务。
- 部署基础设施。
- 硬件实验和板端调试记录。
- 内部日志、工作记录、本地证据包和私有现场配置。

仓库保留历史提交，用于追溯项目演进。历史文件不应视为仍在维护的 API、受支持模块或可直接发布的资产。

## 维护规则

新工作默认保持桌面端聚焦。除非维护者明确决定恢复其他产品面，否则不要重新引入 Web、移动端、后端或基础设施目录。如果未来恢复某个模块，应作为具备独立 CI、用户文档、维护归属和安全审查的明确模块重新加入。
